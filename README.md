# [Odin Recipes](https://geon-youn.github.io/odin-recipes/)

This project uses HTML to create a simple website hosting several recipes that will improve your life. 

Elements of HTML used:
- Ordered and unordered lists (`ul`, `ol`, `li`).
- Anchors (`a`).
- Images (`img`).
- Headers and paragraphs (`h`, `p`)
